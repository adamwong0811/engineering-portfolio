# Reports and Evidence

This folder is reserved for report PDFs, selected figures, screenshots, and measurement images.

For a public GitHub portfolio, I recommend uploading selected evidence rather than every full report by default:

- Keep the main project pages concise and readable.
- Export important plots and photos into `assets/`.
- Add full PDFs only when the file size is reasonable and the report does not expose private teammate details, lecturer details, or internal coursework information that should not be public.

## Suggested Attachments

| Project | Recommended Evidence |
| --- | --- |
| Headphone amplifier | Schematic, PSpice gain plot, THD/FFT plot, breadboard photo, KiCad PCB screenshot |
| Programmable oscillator | PSpice schematic, waveform plots, frequency-vs-code table |
| Digital logic | Breadboard photos, CircuitVerse screenshots, truth tables |
| Verilog | Source files, testbenches, EDA Playground links, waveform screenshots |
| Regenerative feedback | Schmitt trigger hysteresis plot, astable waveform, monostable pulse waveform |
| SECOM analytics | Colab notebook, confusion matrices, model comparison chart |
| NMOS fabrication | Cleanroom photos, oxide thickness screenshots, probe station photo, I-V plots |
| Differential amplifier | PSpice schematic, differential/common-mode transient plots |
| Current mirror | DC sweep plots and current mirror schematic |
| BJT vs FET | Multisim schematics and theory/simulation/measurement table |
| EMI coupling | CADFEKO model screenshot and POSTFEKO frequency-response plot |
| 9 V regulator | Stage-by-stage oscilloscope waveforms and prototype photo |
