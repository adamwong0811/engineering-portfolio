## Project Summary

This laboratory sequence covered the physical implementation of core digital logic systems. It progressed from Boolean expression simplification and universal-gate implementation to arithmetic logic and sequential memory elements.

The work helped connect textbook truth tables and timing diagrams to real breadboard wiring, IC pinouts, propagation behavior, and measurement/debugging.

## Technical Scope

- Simplified Boolean logic using Karnaugh maps and Boolean algebra.
- Implemented logic circuits using NAND/NOR universal gate structures.
- Built and tested a 4-bit parallel adder/subtractor system using SN7483 and SN7486 ICs.
- Constructed sequential memory structures including SR latches, gated latches, D flip-flops, and JK flip-flops.
- Compared observed hardware behavior with truth tables, timing diagrams, and simulation.

## What I Learned

- How to translate a minimized Boolean expression into a physical gate-level circuit.
- Why pin mapping, common ground, and debounced/clear input states matter in breadboard logic.
- How combinational and sequential circuits differ in debugging method.
- How propagation delay and unexpected state behavior appear in real hardware.
