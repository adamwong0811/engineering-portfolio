## Project Summary

This self-directed project transferred digital-logic knowledge from breadboard circuits into RTL design. The exercises focused on writing synthesizable Verilog modules, creating testbenches, and verifying waveform behavior in simulation.

## Implemented Designs

| Design | Concept Practiced |
| --- | --- |
| 4-bit combinational adder | Combinational arithmetic and module I/O |
| D flip-flop with synchronous reset | Edge-triggered sequential logic and non-blocking assignments |
| 4-bit synchronous counter | State transition, clocked updates, reset behavior |
| Testbenches | Functional verification and waveform interpretation |

## What I Learned

- How HDL differs from software programming: code describes hardware structure and timing.
- Why non-blocking assignments are important for clocked sequential logic.
- How to validate simple RTL blocks with testbenches before hardware deployment.
- How breadboard-level digital logic maps into FPGA-style design thinking.
